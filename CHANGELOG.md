# Changelog

Все значимые изменения в проекте Trading Analyzer будут документированы в этом файле.

## [6.0.2] - 2024-12-19

### 🆕 Добавлено
- **БЛОК с предыдущего дня** - возможность формировать ценовой диапазон начиная с предыдущего торгового дня
  - Checkbox в настройках для активации функции
  - Автоматическая корректировка расчетов при активации
  - Полезно для стратегий с использованием ночных сессий

- **Пропуск дней с красными новостями** - автоматическое исключение дней с важными новостями
  - Проверка всего календарного дня (00:00-23:59)
  - Фильтрация по выбранным валютам
  - Работает только с high impact новостями

### 🔧 Изменено
- Обновлен `data_processor.py`:
  - Добавлен параметр `from_previous_day` в метод `get_block_range()`
  - Новый метод `has_high_impact_news_in_day()` для проверки новостей за весь день

- Обновлен `analyzer.py`:
  - Поддержка параметра `from_previous_day` при формировании БЛОКА
  - Проверка и пропуск дней с красными новостями в `analyze_day()` и `analyze_period()`

- Обновлен UI в `app.py`:
  - Новые checkbox для обеих функций
  - Сохранение новых параметров в настройках
  - Информационные сообщения для пользователя

## Формат версий

Проект следует [Semantic Versioning](https://semver.org/):
- **MAJOR** версия - несовместимые изменения API
- **MINOR** версия - новая функциональность с обратной совместимостью  
- **PATCH** версия - исправления с обратной совместимостью
