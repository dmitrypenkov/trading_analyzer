# Statistical Analyzer для Trading Analyzer v6.0

## 📊 Описание
Статистический анализатор торговых данных с фокусом на R-метрики и анализ серий сделок.

## 🎯 Основные возможности

### 1. Анализ серий сделок
- Подсчет максимального количества TP/SL подряд
- Группировка по периодам: весь период, год, месяц
- Режим игнорирования BE/NO_TRADE в сериях
- Распределение длин серий

### 2. Rolling Window анализ
- Настраиваемый размер окна в торговых днях
- Расчет суммарного/среднего/мин/макс R за окно
- Проверка достижения TARGET_R и DRAWDOWN_R
- Визуальное распределение результатов окон

### 3. Экстремальные значения
- Максимальные и минимальные R-результаты
- Группировка по периодам: всего, год, месяц, rolling window

### 4. Анализ типов входов
- Статистика по каждому entry_type
- Распределение результатов (TP/SL/BE/NO_TRADE)
- Средний и суммарный R по типам

### 5. Анализ направлений
- Сравнение Long vs Short
- Win rate и средний R по направлениям

### 6. Временные паттерны
- Лучшие/худшие часы для входа
- Эффективность по дням недели
- Сезонность (анализ по месяцам)
- Среднее время удержания позиций

## 📋 Формат входных данных

CSV файл должен содержать следующие колонки:
- `date` - дата сделки (YYYY-MM-DD)
- `weekday` - день недели
- `entry_type` - тип входа
- `direction` - направление (Long/Short)
- `entry_time` - время входа (HH:MM)
- `exit_time` - время выхода (HH:MM)
- `result` - результат (TP/SL/BE/NO_TRADE)
- `r_result` - результат в R-единицах

## ⚙️ Настраиваемые параметры

### В интерфейсе
- **TARGET_R** - целевая прибыль в R (по умолчанию: 5.0)
- **DRAWDOWN_R** - критическая просадка в R (по умолчанию: -10.0)
- **ROLLING_WINDOW_DAYS** - размер окна в днях (по умолчанию: 10)
- **IGNORE_BE_NO_TRADE** - режим учета BE/NO_TRADE в сериях

### Фильтры данных
- Диапазон дат для анализа
- Фильтр по типам входов (entry_type)
- Фильтр по направлениям (direction)

## 📊 Выходные данные

### Отображение в интерфейсе
1. **Метрики и статистика** - ключевые показатели
2. **Таблицы** - детальные данные по каждому разделу
3. **Распределения** - визуальное представление данных

### Экспорт
- **CSV** - все таблицы и детальные данные
- Отдельные файлы для каждого типа анализа

## 📈 Интерпретация результатов

### Серии сделок
- Показывает склонность стратегии к последовательным выигрышам/проигрышам
- Помогает оценить психологическую нагрузку (длинные серии убытков)

### Rolling Window
- Оценка консистентности стратегии
- Вероятность достижения цели за фиксированный период
- Риск критической просадки

### Временные паттерны
- Выявление оптимального времени для торговли
- Определение сезонности в результатах

## 🔍 Примеры использования

### Базовый анализ
1. Загрузите CSV файл
2. Установите TARGET_R = 5.0 и DRAWDOWN_R = -10.0
3. Выберите ROLLING_WINDOW_DAYS = 10
4. Изучите результаты и экспортируйте отчет

### Анализ конкретного периода
1. Используйте фильтр дат
2. Сравните результаты разных периодов
3. Выявите изменения в эффективности

## 📝 Примечания
- Торговые дни = дни с данными в CSV (не календарные)
- R-единица = размер одного стоп-лосса
- Все расчеты основаны на готовых r_result из CSV
